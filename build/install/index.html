Установка Sass

Существует несколько способов начать использовать Sass:

  1. Приложения

    Есть множество приложений под Windows, Mac и Linux, которые помогут в освоении и работе с Sass. Большинство из них бесплатные, но некоторые из них все-таки платные (но это того стоит).

    • CodeKit (Платно)
    • Compass.app (Условно-бесплатно)
    • Hammer (Платно)
    • Koala (Бесплатно)
    • LiveReload (Условно-бесплатно)
    • Mixture (Бесплатно)
    • Prepros (Условно-бесплатно)
    • Scout (Бесплатно)

  2. Командная строка

    Linux
    Если Вы используете операционную систему Linux, то в первую очередь Вам необходимо установить Ruby. Установить Ruby можно через пакетные менеджеры apt, rbenv или rvm. 
    sudo su -c "gem install sass"
    Windows
    Перед тем, как начать пользоваться Sass Вам необходимо установить Ruby. Самый быстрый способ установить Ruby на Windows — это использовать установщик Ruby. Эта установка в один клик поможет Вам приступить к работе практически мгновенно.
    Установщик также установит командую строку Ruby — приложение, которое позволит использовать все библиотеки Ruby.
    Mac
    Если вы предпочитаете командную строку нежели приложения, то настройка Sass для работы занимает считанные минуты. Так как Sass зависим от Ruby, а Вы пользователь Mac, поздравляем, Ruby уже предустановлен по-умолчанию.
    Установка Sass

    Это самый быстрый способ, чтобы начать использовать Sass с помощью командной строки:

    1. Откройте терминал или командную строку. На Mac Terminal.app установлен по умолчанию и находится в папке «Программы». В Windows запустите команду `cmd`.

    2. Установка Sass. Ruby использует gem для управления различными пакетами такими как Sass. В открытом терминале наберите команду: 

      gem install sass

      Таким способом можно установить Sass и любые другие пакеты. Это просто магия =). Если у Вас отображается ошибка, в которой говорится о том, что нужно использовать команду sudo для установки пакета Sass, то введите в терминале: 

      sudo gem install sass

    3. Наличие пакета. Теперь у Вас должен быть установлен пакет Sass, но если хотите Вы можете проверить это. В терминале наберите команду: 

      sass -v

      В ответ вы должны получить Sass 3.4.16 (Selective Steve). Поздравляем! Sass успешно

      установлен.

    4. Развлекайтесь. Если Вы новичок в Sass, то для Вас мы подготовили много ресурсов, которые помогут быстро изучить Sass.
